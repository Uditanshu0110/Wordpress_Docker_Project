== Installation ==
1. Primary: == Installation ==
	- In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
	- Type in Eventsia in the search form and press the 'Enter' key on your keyboard.
	- Click on the 'Activate' button to use your new theme right away.
	- Go to https://themefreesia.com/theme-instruction/eventsia/ for a guide on how to customize this theme.
	- Navigate to Appearance > Customize in your admin panel and customize to taste.

2. Alternate: Unzip the template file (eventsia.zip) that you have downloaded. Via FTP, upload the whole folder (eventsia) to your server and place it inside the http://yourdomain.com/wp-content/themes/folder(eventsia). Do not change directory name. The template files should thus be here now: http://yourdomain.com/wp-content/themes/eventsia/index.php (for example).
	- Log into your WP admin panel and click on the Appearance -> Theme tab. Now click on the Eventsia theme to activate it.
	- Want to enjoy Eventsia feature. Go to  Dashboard -> Appearance -> Customize and play with Eventsia Options
	- The theme is offered free of charge. If someone asked money for it, someone just tricked you.

== Background Color/ Image not Working?
	- Background Color/ Image works with Boxed and Small Boxed Layout only. If your site is displaying Full Width Layout, then Background Color/ Background Image will not work. This theme is designed this way. Go to Dashboard > Appearance > Customize > Theme Options > Design Layout > Layout Options
